§ 02 / Manifesto

Less spectacle.
More circulation.

A working set of beliefs about what biotech, materials, and software ought to do — and what they ought to refuse.

I treat waste streams, field data, and code as the same kind of raw material — things to be looped back into a system that closes on itself, not opens onto a landfill.

I'm a B.Tech biotechnologist, but the projects on this page are deliberately interdisciplinary. Some are wet lab. Some are full-stack. Most are both — because the interesting problems don't sit politely inside one syllabus.

Every project here begins with a specific failure mode in an existing system: middlemen extracting value from farmers, plastic that no machine wants, bitumen roads dissolving in monsoon. Every solution begins with the question "what's already nearby that we're throwing away?"
